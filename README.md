# Password Strength Analyzer

## Description
This project evaluates the strength of a user-entered password based on security rules.

## Features
- Checks password length
- Checks uppercase, lowercase, numbers, special characters
- Provides strength score
- Suggests improvements

## Technologies Used
- Python
- Regular Expressions

## How to Run
1. Open terminal in VS Code
2. Run: python main.py
3. Enter a password

## Outcome
This project helps understand password security concepts.s
